# bb-mb_files — Maria Rita | Turnê Amor e Música

Repositório de apoio usado para reunir os materiais de mídia (capas e faixas de áudio) relacionados à turnê **Amor e Música**, da cantora Maria Rita. Serve como repositório central para facilitar o acesso a esses arquivos durante a produção/organização do show.

| `Opera Instantâneo_2026-07-15_104414_www.google.com.png` | Foto de divulgação da artista |

| `encontros-e-despedidas.mp3` | Faixa "Encontros e Despedidas" |

## Como usar

1. Baixe ou visualize os arquivos diretamente pela pasta do repositório.
2. As imagens podem ser usadas como capas/material de divulgação da turnê.
3. Os áudios podem ser usados como referência sonora para ensaios e apresentações.

## Colaboração

Este repositório é mantido em colaboração. Para contribuir com novos arquivos, abra um Pull Request com as alterações propostas.
